# AI Chat Assistant

An intelligent chatbot powered by OpenAI GPT-4 with custom knowledge base integration and real-time conversation capabilities.

## Overview

This project demonstrates the power of modern AI APIs in creating conversational interfaces that can understand context, maintain conversation history, and provide intelligent responses based on custom knowledge bases.

## Technical Implementation

### Backend (Python/FastAPI)

```python
from fastapi import FastAPI, WebSocket
from openai import OpenAI
import json

app = FastAPI()
client = OpenAI()

@app.websocket("/chat")
async def websocket_endpoint(websocket: WebSocket):
    await websocket.accept()
    
    while True:
        try:
            message = await websocket.receive_text()
            response = client.chat.completions.create(
                model="gpt-4",
                messages=[{"role": "user", "content": message}],
                stream=True
            )
            
            for chunk in response:
                if chunk.choices[0].delta.content:
                    await websocket.send_text(chunk.choices[0].delta.content)
                    
        except Exception as e:
            await websocket.send_text(f"Error: {str(e)}")
```
